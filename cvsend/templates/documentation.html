Steps to get Successful mail drops

Enter email address for which you have generated token

Enter 16 digits token generated from mail service provider under token field

Enter the subject which you have to fill in each mail, you can use the keyword to replace it with designation you are going to apply

Enter the body of mails, here too you can use the keyword to replace it with designation you are going to apply

Upload Excel(.xlsx) file with the format described under Prep .xlsx file

Upload your resume at the 2nd upload button. Allowed formats were .pdf, .docx & .odt

Finally click Submit button and wait for mails to send

How to Generate "token"?

Turn on 2-Step Verification

With 2-Step Verification, also called two-factor authentication, you can add an extra layer of security to your account in case your password is stolen. After you set up 2-Step Verification, you can sign in to your account with:

  1. Your password
  2. Your phone

Click on top-right profile icon, once after Login to your Gmail Account Google Account

Select Manage your Google Account

Under Security -> 2-step verification

Select Your preferred method for verification and select continue

Select 2-step Verification -> Select App password

under the window select device as others and name it as your wish

You should now generated with 16 digit token. Fill the token under Token field.

Guide to create .xlsx file

To do this you need to open Excel or Google sheet, make the heading(1st row) as designation and email in two separate cells

Fill the corresponding rows with appropriate datas. Be ensure that you have filled email id's under email heading and designation you are going to apply under designation heading

If it's a Excel save it as usual

If using google sheet select File -> Download -> Microsoft Excel(.xlsx)

Once done with this Steps upload the file using Excel Upload button.
